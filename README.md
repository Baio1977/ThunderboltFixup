ThunderboltFixup
===================

An open source kernel extension providing patches for AppleThunderboltNHI.kext.

#### Boot-args
- `-tbfixupoff` disables kext loading
- `-tbfixupdbg` turns on debugging output
- `-tbfixupbeta` enables loading on unsupported osx
